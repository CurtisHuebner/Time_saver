Block-list:

Site blocking works with whitelisted times and blacklisted sites. For instance to block "www.evil.com" and "www.distraction.com" from 1:30AM Monday to 4:37PM Friday and any site that has "fun" in the url on Sunday, the filter would be: 

{
    "times": [
        {
            "endDay": 5,
            "endHour": 16,
            "endMinute": 37,
            "startDay": 1,
            "startHour": 1,
            "startMinute": 30,
            "urls": [
                "www.evil.com",
                "www.distraction.com"
            ]
        },
        {
            "endDay": 1,
            "endHour": 0,
            "endMinute": 0,
            "startDay": 0,
            "startHour": 18,
            "startMinute": 0,
            "urls": [
                "fun"
            ]
        }
    ]
}

It's a terrible UI. Deal with it.
